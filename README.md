2x Schiff
   - kann beschädigt

    4x Insel
        je Schwierigkeit 0-2
        je 1 Aufgabe
        kann +1 Berry gewinnen
    2 x Reisepunkte
        4-6 x Reiseproben

4x Modifikator D6 (2x0 2x+1 2x-1)
